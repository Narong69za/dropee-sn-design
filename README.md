# Dropee SN Design

Automation Engine for Dropee Tasks

Developed by SN DESIGN STUDIO

---

## Overview

Dropee SN Design is an automation engine designed to automate tasks inside the Dropee ecosystem.

The system supports multi-account automation and lightweight execution in Termux environments.

---

## Features

• Auto Task Engine  
• Auto Reward System  
• Tap Automation  
• Multi Account Manager  
• Lightweight Termux Compatible  
• Modular Python Architecture  

---

## Installation

pkg update && pkg upgrade -y

pkg install git -y
pkg install python -y

git clone https://github.com/Narong69za/dropee-sn-design.git

cd dropee-sn-design

pip install -r requirements.txt

---

## Configure Accounts

nano accounts/accounts.txt

Add token per line

---

## Run

python main.py
